# **A Minecraft Report Plugin**

> ## Features / Command
> 
> /report \<player\> \<reason\>
> 
> `Report player with reason`

------------------------------

> ## Config
> 
> ```
> webhookURL: URL
> cooldowns: 600
> ```
> 
>> Change URL to your discord webhook url.
>> 
>> Change 600 to your cooldown time (seconds) for command.
